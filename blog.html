What is DOM and what is the purpose of DOM?

Document Object Model (DOM) is used to manipulate web pages. Its purpose is to provide a structured representation of the HTML document, which can be accessed and modified with JavaScript.

How will you select HTML elements using DOM? Name the DOM methods?

We can select HTML elements using the DOM in 5 methods such as:
1. getElementsByTagName() : We can select elements with tag name.
2. getElementById() method: We can select an element by its ID.
3. getElementsByClassName() method: We can select elements with a class name.
4. querySelector() method: We can select the first element that matches a CSS selector.
5. querySelectorAll() method: We can select all the elements that match a CSS selector.

What is event bubble?

Event bubbling is a way in which events in HTML are first captured and handled by the innermost element, then propagate up through its parent elements in the DOM tree.

What is a callback function and why will you use it?

A callback function is a function that is passed as an argument to another function and is called by that function once it is ready to return a result. We can use it to as a common function so that we do not have to write same kind of functions.
